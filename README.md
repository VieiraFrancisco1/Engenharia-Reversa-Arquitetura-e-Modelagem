Descrição da Atividade

Esta atividade tem como objetivo aplicar conceitos de Engenharia Reversa, Arquitetura e Modelagem de Software a partir da análise de um sistema real em produção.

O sistema analisado foi um site de pedidos online de uma pizzaria, no qual foram observadas suas funcionalidades, organização, arquitetura e possíveis padrões de projeto, sem acesso ao código-fonte.

A análise foi realizada com base no comportamento do sistema, utilizando conceitos como:

Arquitetura de Software
Design de Software
Engenharia Reversa
Modelagem UML
Padrões de Projeto
Estrutura do Repositório

O repositório está organizado da seguinte forma:

analise.md: análise do sistema real
comparacao.md: comparação com sistema didático
arquitetura.md: proposta de arquitetura

Objetivo

O objetivo principal é demonstrar a capacidade de analisar, modelar e propor melhorias em um sistema real, utilizando boas práticas de desenvolvimento de software.
